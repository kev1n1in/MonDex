# Getting Started

from npm:

1.Clone the repo:

```bash
git clone git@github.com:kev1n1in/MonDex.git
```

2.Install NPM packages:

```bash
npm install
```

3.Run the development server:

```bash
npm run dev
```

# Usage

Use Mondex to find your Pokoemon or Digimon
